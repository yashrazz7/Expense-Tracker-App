\# Expense Tracker App



A simple, lightweight, and professional Web Application built to track daily expenses efficiently. This project demonstrates a functional full-stack setup using a Java backend and a responsive frontend.



\## Features

\- \*\*Add Expenses:\*\* Easily add items with their Name, Amount, and Category.

\- \*\*Track Expenses:\*\* View all your expenses in a clean, organized table.

\- \*\*Persistent Storage:\*\* Your data is automatically saved to an `expenses.txt` file, ensuring no data loss when the server restarts.

\- \*\*Delete Entries:\*\* Remove any incorrect or unwanted expenses with a single click.



\## Technologies Used

\- \*\*Backend:\*\* Java (using `com.sun.net.httpserver`)

\- \*\*Frontend:\*\* HTML5, CSS3, JavaScript (Fetch API)

\- \*\*Data Storage:\*\* Local Text File (`expenses.txt`)
